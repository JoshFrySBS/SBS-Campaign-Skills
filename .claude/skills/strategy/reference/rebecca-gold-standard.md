# Gold-Standard Strategy Document — Rebecca Hannah (DFY Reference)

This is the canonical example of a complete SBS DFY strategy document. The `/strategy` skill must produce documents that match this structure, depth, and tone.

**Why Rebecca?** This was the first DFY strategy run end-to-end with the full strategy companion. Every section hit 88%+ confidence, the campaigns built from it are live, and the document has been used as the build brief by the operator. It's the proven shape of a working DFY strategy.

**How to use this reference:**
- Match the section order and the section structure exactly
- Match the depth — every section has tables, not just bullets; quotes in their own voice; specific numbers; named avatars
- Match the tone — direct, declarative, peer-to-peer; no hedging on the strategy itself even though the copy uses hedge language
- Lift the header pattern, executive summary table, confidence scoring table, "What They're Thinking" quote pattern, the trigger tables (10A copywriting / 10B research), and the Strategic Angles table
- Do NOT copy Rebecca-specific content (NZ market, AGM, $4B KiwiRail, etc.) — adapt the **shape**, replace the **substance** with the new client's data

**Critical structural elements every strategy doc must hit:**
1. Header block + Executive Summary (one-line strategy + Campaign Snapshot table + Confidence Scores table)
2. ICP avatars with named characters (e.g. "Peter the CEO", "Sarah the EA") + company profile + key characteristic quote
3. Offer in plain English (italicised pull-quote) + Service Details table + Proof table + Transformation before/after table
4. Core Pains structured as numbered pains with "What They're Thinking" first-person quotes per pain
5. Desired Outcomes split into Tangible + Emotional + Dream Outcome
6. Buying Signals & Triggers with Primary trigger table + Secondary triggers table + Campaign Priority Stack
7. Objections per ICP as Objection / Reality / How to Handle table + a Key Insight pull-quote
8. Differentiator table + 30-second pitch (italicised pull-quote)
9. TAM & Campaign Math (Market Size table + Funnel Math table + Reality Check pull-quote)
10. Tone of Voice per ICP with example lines + Dial Up/Dial Down table
11. Trigger Suggestions split into 10A Copywriting (no enrichment) and 10B Research (Clay enrichment) — with hooks in client voice
12. Strategic Angles table (Angle / Lead With / Hook / Best Trigger)
13. Poke-the-Bear Lines (numbered, per ICP)
14. FOMO Angles table (FOMO Angle / Core Idea / Sample Line)
15. Personalisation Logic (14A No-Enrichment table + 14B Enriched table with variables)
16. Lead Sourcing — Clay Workflows (DFY only) — step-by-step
17. Clay Table Structure (DFY only) — column-by-column
18. Next Steps (Immediate Actions + Campaign Launch Order + Success Metrics)

# **SECTION 15: LEAD SOURCING - CLAY WORKFLOWS**

## **15.1 Campaign 1: AGM Campaign (Primary)**

**Objective: Target CEOs and EAs at NZ companies 8-12 weeks before their AGM.**

### **Step 1: Build the Company List**

* Source: NZX-listed companies (easiest to find AGM dates)
* How: Scrape NZX company list OR manually compile top 200-300 NZX companies
* Add columns: Company Name, NZX Ticker, Industry, Revenue

### **Step 2: Enrich with AGM Dates**

* Clay enrichment: Use "Scrape Website" action on each company's investor relations page
* AI prompt: "Find the next AGM date for this company. Return in format DD/MM/YYYY."
* Add columns: agm_date, weeks_until_agm (calculated field)

### **Step 3: Filter for Timing**

* Filter: weeks_until_agm between 8 and 12
* Refresh weekly to catch new companies entering the window

### **Step 4: Enrich with CEO Contacts**

* Clay: "Find People at Company" > Filter: Title contains "CEO" OR "Managing Director"
* Add columns: ceo_first_name, ceo_last_name, ceo_email, ceo_linkedin
* Use Clay's waterfall email finding (or plug in FindyMail, Prospeo)

### **Step 5: Enrich with EA Contacts**

* Clay: "Find People at Company" > Filter: Title contains "Executive Assistant" OR "Chief of Staff"
* Add columns: ea_first_name, ea_last_name, ea_email, ea_linkedin

### **Step 6: Push to Instantly**

* Create two campaigns: "Rebecca AGM - CEOs" and "Rebecca AGM - EAs"
* Map variables: {{first_name}}, {{company}}, {{agm_date}}, {{weeks_until_agm}}
* Refresh cadence: Weekly
